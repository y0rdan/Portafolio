
# Python Security & Utility Projects Collection

## Overview

This repository contains a collection of Python-based cybersecurity, utility, and learning projects. Each folder is a standalone module focused on a specific domain such as offensive security tools, defensive monitoring, automation, or general Python development practice.

The goal of this repository is to demonstrate practical skills in:
- Cybersecurity fundamentals (scanning, detection, analysis)
- Secure coding practices
- Automation and scripting
- Basic game and utility development in Python

---

## Repository Structure

### Encrypted/
Contains scripts and utilities related to encryption and decryption.
Typical use cases:
- File encryption/decryption
- Basic cryptographic operations
- Learning symmetric/asymmetric encryption concepts

---

### File_integrity_Monitoring_Tool/
A file monitoring system that tracks changes in files and directories.

Key features:
- Hash-based integrity checking (e.g., SHA-256)
- Detection of file modifications, deletions, or additions
- Useful for endpoint protection and system auditing

---

### Log Analyzer/
A tool designed to parse and analyze system or application logs.

Key features:
- Log parsing and filtering
- Pattern detection (errors, anomalies, suspicious activity)
- Basic reporting for security or debugging purposes

---

### OPP Cybersecurity IDS/
An Object-Oriented Python Intrusion Detection System (IDS).

Key features:
- Network or log-based intrusion detection logic
- Rule-based or behavior-based detection
- Modular OOP design for extensibility

---

### Password Checker/
A password security validation tool.

Key features:
- Strength analysis (length, complexity, entropy estimation)
- Common password detection
- Feedback for improving password security

---

### Port Scanner/
A basic network scanning utility.

Key features:
- TCP port scanning on target hosts
- Detection of open/closed ports
- Used for network reconnaissance in controlled environments

---

### PyGame Projects/
Collection of simple games and interactive applications built using Pygame.

Key features:
- Game loop implementation practice
- Event handling and rendering
- Basic game mechanics development

---

### Web Crawler/
A simple web crawling and scraping tool.

Key features:
- URL traversal and link extraction
- HTML parsing
- Data collection from web pages (for learning and research purposes only)

---

## Notes

- All projects are intended for educational and lab use only.
- Some tools simulate real-world security techniques and should only be run in authorized environments.
- Code is modular and can be extended for more advanced implementations.
